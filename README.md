Este projeto foi desenvolvido com o objetivo de apresentar o artigo **"A Burocratização da Adoção no Brasil"** em formato de página web.

A ideia foi criar uma experiência de leitura mais agradável do que um documento tradicional, utilizando recursos de HTML, CSS e JavaScript para tornar a navegação mais dinâmica, organizada e responsiva.

Além do conteúdo acadêmico, o projeto busca transmitir as informações de maneira clara, valorizando tanto a estética quanto a experiência do usuário.

---

## 💡 Objetivo

O principal objetivo deste projeto é apresentar, de forma visual e interativa, uma reflexão sobre os desafios enfrentados no processo de adoção no Brasil.

Durante o desenvolvimento, foram utilizados recursos modernos do desenvolvimento web para construir uma página organizada, intuitiva e adaptável a diferentes dispositivos.

---

## 🛠️ Tecnologias utilizadas

O projeto foi desenvolvido utilizando apenas tecnologias da Web:

- HTML5
- CSS3
- JavaScript

---

## 📂 Estrutura do projeto

```text
artigoDeNina/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

### 📄 index.html
Responsável por toda a estrutura da página, contendo os textos, seções e organização do artigo.

### 🎨 style.css
Arquivo responsável pela aparência do projeto, incluindo cores, tipografia, responsividade, animações e layout.

### ⚙️ script.js
Contém toda a parte de interação da página, como animações durante a rolagem, efeitos visuais e atualizações dinâmicas dos elementos.

### 📘 README.md
Arquivo de documentação do projeto.

---

## ✨ Funcionalidades

Entre os principais recursos implementados estão:

- Layout responsivo;
- Estrutura utilizando HTML semântico;
- Animações durante a rolagem da página;
- Indicador visual das seções;
- Barras de progresso animadas;
- Scroll suave;
- Design inspirado em documentos jurídicos.

## ▶️ Como visualizar

Para executar o projeto basta abrir o arquivo **index.html** em qualquer navegador.

Caso utilize o Visual Studio Code, recomenda-se utilizar a extensão **Live Server**, permitindo visualizar as alterações em tempo real durante o desenvolvimento.

## 📚 Conteúdo apresentado

O artigo está organizado em diferentes seções, abordando temas como:

- O conceito de adoção;
- A burocracia no processo de adoção;
- Os impactos da demora;
- Dados estatísticos do Conselho Nacional de Justiça (CNJ);
- O perfil dos pretendentes à adoção;
- Reflexões sobre possíveis melhorias no sistema.

---

## 📖 Fonte das informações

As informações apresentadas foram baseadas em estudos sobre o processo de adoção no Brasil e em dados divulgados pelo **Conselho Nacional de Justiça (CNJ)**.
